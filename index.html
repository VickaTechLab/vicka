Temporizador

Selecciona la opción "Iniciar temporizador" y te pediremos el tiempo en segundo para hacer la cuenta regresiva.

El temporizador recibe valores entre 0 y 60 para que no estés más de 1 minuto esperando a que termine la prueba :D

0 segundos

Tareas

Selecciona la opción "Cargar tareas" y te pediremos que ingreses las tareas que necesitas realizar para cumplir tus objetivos. Para finalizar la carga de tareas, presiona ESC o Cancelar.

Historias de Usuario

Selecciona la opción "Agregar historia de usuario" para agregar una nueva historia de usuario al proyecto y "Ver historias de usuario".
